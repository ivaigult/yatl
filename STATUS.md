# Support status
## Types
### Atomic Types
- [x] Boolean
- [ ] Numbers
  * [ ] Complex
  * [x] Real
  * [ ] Rational
  * [ ] Integer
- [x] Strings
- [ ] Characters
- [x] Symbols
### Compound Types
- [x] Lists
- [ ] Pairs (`(a . b)` doted pairs are unsupported)
- [ ] Vectors
## Special Forms
- [x] Lambda expressions (no TCO)
- [ ] Lexical bindings
  * [x] `define`
  * [x] `set!`
  * [x] `let`
  * [x] `let*`
  * [x] `fluid-let`
  * [ ] `letrec`
- [x] Quoting
- [ ] Quasi quoting
- [ ] Conditionals
  * [x] `cond`
  * [ ] ...
- [x] begin
- [ ] iterations
- [ ] macroses
